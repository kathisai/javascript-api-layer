Java Script API Layer

How to run in your machine

--Prerequisites

  • npm (node)
  • http-server

Steps -Clone project from https://gitlab.infotraxsys.com/prathapk/javascript-api-layer.git -At root, run npm install. This will install all dependencies in package.json file -At root, run http-server . This will start localhost server like below

Starting up http-server, serving .
Available on:
 http://10.10.3.237:8080
 http://172.20.40.23:8080
 http://127.0.0.1:8080

-Open http://localhost:8080/ in Google chrome.

You may face Cross-Origin issue as we are reaching different servers from localhost. you can see issue in console like below

Cross-Origin Request Blocked: The Same Origin Policy disallows reading the remote resource at http://dev-evo-dtsrest.myvoffice.com/index.cfm/v1/commissions/qualifications/volumes/undefined/function(t)%7Bconsole.log(%22Data%20:%20%22,t)%7D?0=function(t)%7Bconsole.log(%22Error+:+%22,t)%7D. (Reason: CORS header ‘Access-Control-Allow-Origin’ missing).[Learn More]

#How to fix Cross-Origin

Open Google Chrome with below command C:\Program Files (x86)\Google\Chrome\Application>chrome.exe --disable-web-security --user-data-dir This will disable web security for us to make calls from different servers from localhost. Above command automaticall open Chrome istance for us to use.

-Now we can use API from in html or js file as mentioned below

Inside JS files

import Api from 'frontend_sdk'; 

Api.getRanks();
Api.getVolumes(3); 
Api.getPersonalVolume(3);
